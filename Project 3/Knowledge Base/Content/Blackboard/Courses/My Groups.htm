My Groups

The Groups drop-down menu is located at the bottom of the Course Menu (if your instructor has set up groups). This menu has several features that are

beneficial when working with a team. Below is a list of these features and what they do.

 

Collaboration

You can create a Collaboration Session where other members of your group can join and chat

File Exchange

The File Exchange option is used for the sharing of files between group members. File Exchange is similar to programs like Dropbox and Google Drive.

Group Blog

A blog embedded in blackboard that all members of the group can make posts to.

Group Discussion Board

Members of a group can make posts in this area. Each post can then be replied to by other members of the group in the fashion of a forum.

Group Journal

Journal Entries can be created in this area to monitor progress on an assignment or to build a timeline of group activities.

Group Tasks

Tasks can be created with information on when they are due and how much of a priority the task is.

Group Wiki

A wiki page dedicated to the group, all members of the group can post and make changes.

Group Homepage

Gives an overview of everything group related (collaboration, file exchange, etc.)

Customize Group Banner

Provides the option to change the banner for the Group Homepage

Customize Group Color

Provides the option to change the colors in the Group Homepage
