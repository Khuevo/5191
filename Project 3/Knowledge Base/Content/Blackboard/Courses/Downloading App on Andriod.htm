When installing Mobile Learn on an Android device, every user must allow the app a set of permissions.

Below is a list of these permissions, and what the app actually does with each.

Storage

Allows users to download/upload attachments for forums, blogs, journals, etc.

System tools

Allows for the creation and removal of application shortcuts on the device's home screens.

Phone calls

Allows for push notifications to the device.

Network communication

Necessary in order to communicate with your institution's server and load data from the internet.

Your accounts

Allows users to save their Learn login credentials for future use.

Your social information

When using the Mobile Learn Roster tool, this permission allows the app to search the device's Contacts

list to find any matches between the Roster and your Contacts. Mobile Learn does NOT read your

device's call log, it will only read your contacts if you decide to search for them through the Roster.
