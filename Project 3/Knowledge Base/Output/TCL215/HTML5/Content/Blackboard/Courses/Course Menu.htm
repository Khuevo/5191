The Course Menu acts as a navigation bar so you can quickly view the content you are searching for.

This menu is located on the left side of the screen on any course page (see picture below). At the top of the course menu, the course and section number, the name and the semester you are taking the course are displayed in bold. Next to the course name, there is a house icon that returns you to the course homepage when clicked. Below this, there are links to any tools that are available for the specific course such as:

  • Announcements
  • Course Content
  • Discussions
  • My Grades
  • Messages
  • My Groups

Not all course menus have the same tools available so don’t be worried if you see a Course Menu that is lacking some of the previously mentioned features.
