You are here: EagleConnect > Emails > Clutter

Clutter is a new feature of EagleConnect and Outlook. This tool helps users filter low-priority e-mails. Clutter works automatically after a user turns the

application on. This tool recognizes the e-mails users are mostly likely to ignore and places these e-mails in the Clutter folder. Users may also turn off

the tool.

 

The following sections describe how to turn Clutter on and off as well as how to move e-mail to and from Clutter.

 

Turning Clutter On/Off

1. Select the Settings menu

2. Click Options

The following menu should appear

 

3. Select Clutter

 

4. Select your desired preference

 

 

How to Move E-mails to Clutter

1. Right-click on the e-mail you would like to move

The below dropdown menu should appear: 

 

2. Click Move to Clutter.

 

How to Remove E-mails from Clutter

1. Select the Clutter menu as shown below

 

2. Right-click on the e-mail you would like to remove from Clutter

The below dropdown menu should appear.

 

3. Select Move to Inbox
