One of the most important features on MyUNT is the ability to register for classes.

Searching for a Class

1. Click the Enrollment button on the navigation bar (located on the left of the MyUNT page)

  • A menu will drop down with the following options:
    • Enroll/Add/Drop
    • Swap Classes
    • Edit Classes
    • Enrollment Dates
    • Enrollment Verification

2. Click the Enroll/Add/Drop button

3. Click the Search tab

4. Select University of North Texas from the Institution menu

5. Select Undergraduate from the Career menu

6. Select the appropriate term for which you are applying from the Term menu

7. Select the subject of the course you are looking for or type in the course abbreviation.

8. (optional) Insert the course number

9. Click search

10. Click on the name of a course that interests you to read more about the class. Information about the class capacity, the location of the class and the Class Number is provided here.

If you want to register for a class you find, make sure to write down the Class Number because it is important when enrolling in the course.

 

Enrolling in a Class

1. Click the Enrollment button on the navigation bar (located on the left of the MyUNT page)

2. Click the Enroll/Add/Drop button

3. Select the term that you want to register for

4. Click Continue

5. Type the Class number of the course you want to enroll in and click Enter

6. Click Finish Enrolling

7. Check to make sure the class was added successfully.
