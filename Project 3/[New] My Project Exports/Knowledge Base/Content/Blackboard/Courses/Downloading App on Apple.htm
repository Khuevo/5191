The Blackboard Mobile Learn app can be downloaded for iPhone, iPad and iPod Touch. The current version of the application supports iOS 6 and above.

Downloading Blackboard Mobile Learn on an Apple Device

1. Open the App Store

2. Tap the search icon in the bottom right of the screen

3. Search: Blackboard

  • The first application in the list should be Blackboard Mobile Learn ™

4. Tap the +GET button (located directly to the right of the title)
