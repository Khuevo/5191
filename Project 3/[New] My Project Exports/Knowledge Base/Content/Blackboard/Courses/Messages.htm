The Messages menu is used to send messages to other students or to your professor. Messages send in Blackboard are only viewable to others when

they are using Blackboard (the mail is not tied in with your EagleConnect). The Blackboard website describes this service by stating, “Course Messages

are private and secure text-based communication that occurs within a course and among course members. Although similar to email, users must be

logged into the course to read and send messages.”

 

By default there are two folders on the Messages page, Inbox and Sent.

 

Adding Additional Folders:

1. Click the Create Folder button

2. Type the name of your new folder in the Name field

3. Click Submit

 

Sending a Message

1. Click the Compose Message button

2. Click the TO button under the Recipients category

3. Select the name of the person(s) that you want to message and press the > button to move the name of person from the selectable group of people to the Recipients box.

  • You can also double click a person's name to add or remove them from the message
  • If you want to take someone off of the recipients list, select their name and click the < button

4. Type the subject of your message in the Subject field. (This is required to send a message)

5. (Optional) Type the body of your message in the Body field.

a. Text in the Body field can be edited to change the font, font size, list type and text style.

6. (Optional) Attach a file by clicking the Browse button and selecting the file on your computer that you want to send with the message.
